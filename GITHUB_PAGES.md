# Deploy to GitHub Pages

Your site will be available at:
https://djuro-masonicic.github.io/Lux-stay/

## Setup

1. Go to your repository settings: https://github.com/Djuro-Masonicic/Lux-stay/settings/pages
2. Under "Build and deployment", set Source to "GitHub Actions"
3. Push this code
4. Wait ~2 minutes for deployment
5. Your site will be live!

## Manual Deployment

If you want to deploy manually:

```bash
npm run build
```

This creates a static site in the `out/` folder that you can host anywhere.
